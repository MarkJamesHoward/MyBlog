Here are my top 5 CSS Shotcuts - hopefully there is a new and useful one in there for you!


####1. Center items (vertical and horizontal) in CSS Grid

rather than

```css
align-items: center;
justify-content: center;
```
use
```css
place-items: center;
```

####2. Background Color

```css
background-color: red; => background: red;
```
