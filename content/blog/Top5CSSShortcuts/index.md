Here are my top 5 CSS Shotcuts - hopefully there is a new and useful one in there for you!


####1. Center items (vertical and horizontal) in CSS Grid

Rather than

```css
align-items: center;
justify-content: center;
```
use
```css
place-items: center;
```

####2. Border

Rather than
```css
border-width: 1px;
border-style: solid;
border-color: #000;
```
Use:
```css
border: 1px solid #000;
```

####3. Background Color

Rather than
```css
background-color: red;
```
Use:
```css
background: red;
```
####4. Margins

Rather than
```css
margin-top: 10px;
margin-right: 5px;
margin-bottom: 10px;
margin-left: 5px;
```
Use:
```css
margin: 10px 5px 10px 5px;
```
